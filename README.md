This browser extension adds an additional panel to the upload and edit submission pages in Fur Affinity.

This panel allows you to type in a test search query, and it will tell you where your submission would rank in that search with the currently filled out information.

The following is the definition of the equation used to calculate a submission's base ranking score.
![Mathematical equation defining Fur Affinity's base ranking calculation](https://github.com/Kinachi249/fur-affinity-seo-helper/raw/main/readme-images/equation.jpeg "Search Ranking Equation")

Once your submission's information is filled out, you may try different search queries to see where your submission will rank in a live search. You can use this to refine your submission information to help rank your submissions higher on targeted searches.
